# Website_JoseLuis_Estagio
Desenvolvimento de Website para Hipnoterapeuta.

# Descrição
Este projeto consiste no desenvolvimento de um website institucional para apresentação dos serviços clínicos do Hipnoterapeuta José Luís. 
O website foi desenvolvido no âmbito de um estágio curricular e tem como objetivo disponibilizar informação clara sobre os serviços prestados e facilitar o contacto com clientes.
O principal objetivo deste projeto foi desenvolver um website funcional, organizado e visualmente apelativo, aplicando conhecimentos de desenvolvimento web adquiridos durante a formação.


# Funcionalidades
- Apresentação Profissional do Hipnoterapeuta;
- Slider Automático de Imagens na Secção Principal (Hero);
- Secção "Sobre Mim";
- Apresentação das Áreas de Intervenção;
- Apresentação das Hipnoses Clínicas;
- Testemunhos de Clientes;
- Formulário de Agendamento de Consultas;
- Integração Direta com WhatsApp;
- Ligações para Redes Sociais;
- Design Responsivo para Desktop, Tablet e Mobile;


# Tecnologias Utilizadas
. HTML;

. CSS;

. JavaScript;


# Bibliotecas Utilizadas
. Font Awesome (Ícones);


# Estrutura do Website
O website é composto pelas seguintes páginas:

### Home (index.html) – Página Principal do Website;
    - Apresentação da Atividade Profissional;
    - Menu de Navegação;
    - Apresentação do Profissional;
    - Apresentação dos Serviços Disponiveis;
    - Testemunhos de Clientes;


### Agendamento (agendar.html) – Formuláio para Contactar com o Hipnoterapeuta;
    - Formulário de Contacto;
    - Integração com WhatsApp;
